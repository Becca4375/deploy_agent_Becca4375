Ensure required files (attendance_checker.py, assets.csv, config.json, reports.log) are in the current directory before running.
Make the script executable: (chmod +x script.sh)
Run the script: (./script.sh)
Follow the prompts (enter directory name and optional threshold updates).
🔹 Archive Feature (Ctrl + C)
While the script is running, press Ctrl + C
The script will:
Create an archive: attendance_tracker_<name>_archive
Delete the incomplete directory
Exit safely
